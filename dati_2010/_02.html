SDOF system response to Periodic and Non-Periodic Loading

In this class we studied

  • the response of a SDOF system to periodic loads, using a Fourier Series representation of loading, the frequency response function and superposition of effects,
  • the response of a SDOF system to a generic load in terms of system response to infinitesimal impulses composed using superposition of effect by Duhamel Integral,
and finally we discussed an homework exercise about a SDOF system loaded by a triangular load, whose response was to be seeked by means of Duhamel Integral.

Some material used in this class is now available,

  • the slides i used,
  • a ready to print version of the slides,
  • a spreadsheet that computes the response of a system using the Duhamel Integral in a step-by-step implementation,
  • a printout of the above spreadsheet, displaying a graph with the response to different excitations,
  • a short python script that computes and prints the response, written with particular regard to clarity and extensibility.
